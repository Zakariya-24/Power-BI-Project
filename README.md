# Power-BI-Project

# 📊 Power BI Sales Dashboard Portfolio

This repository contains my **Power BI project** from the Data Technician programme. The project focuses on using Power BI to analyse a sales dataset, create interactive visuals, and present clear business insights.

---

## 🚀 Project Overview

The aim of this project was to build a Power BI dashboard that helps users understand sales performance and identify useful business trends.

The dashboard can be used to explore:

* 📈 Sales trends over time
* 🛍️ Product or category performance
* 🌍 Regional or location-based sales patterns
* 💰 Revenue and profit insights
* 👥 Customer or segment performance

---

## 🧰 Tools Used

* Power BI Desktop
* Power Query
* DAX
* Excel / Sales dataset
* Data visualisation techniques

---

## 🔍 What I Found

The Power BI dashboard helps turn raw sales data into clear and useful insights.

It shows which areas of the business are performing well and where improvements may be needed. By using interactive filters and visuals, users can quickly compare sales performance across different categories, time periods and customer groups.

The main finding is that Power BI makes it easier to spot trends, compare results and support better business decisions.

## 💡 Reflection

This project helped me understand how Power BI can be used in a real business setting. Instead of looking through raw data manually, the dashboard makes the information easier to understand and explore.

I learned how important it is to design dashboards that are clear, simple and focused on the user. A good dashboard should not just look nice, but should help people make decisions quickly.
